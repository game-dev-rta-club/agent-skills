# Changelog

Notable user-facing changes are recorded in this file.

## 2.0.0 - 2026-07-17

- Renamed the repository and plugin to `rubber-ducking-skill` while preserving
  the existing GitHub history, release, and old repository URL redirect.
- Existing installations must remove the `agent-skills` plugin before updating
  the marketplace and installing `rubber-ducking-skill` to avoid duplicate
  plugin identities.
- Clarified that `rubber-duck-caller` and `rubber-duck-partner` are the two
  internal roles of one public rubber-ducking product.
- Added the Contributor Covenant 3.0 Code of Conduct.

## 1.0.0 - 2026-07-15

- Added the `rubber-duck-caller` and `rubber-duck-partner` skills.
- Added plugin distribution metadata for Codex and Claude Code.
- Added project-local installation through `gh-linked-skills`.
